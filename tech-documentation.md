# Анализ технической документации ВОЛС-решений на 2025 год

## Решения с ИСЧЕРПЫВАЮЩЕЙ технической документацией

### 1. NextGIS Web (версия 3.1 от 03-02-2025)
**Статус документации: ОТЛИЧНЫЙ**
- Объем: 350+ страниц полного PDF-руководства
- Язык: Русский, с англ. вариантом
- Содержит:
  - 36 основных глав от введения до глоссария
  - Полное руководство пользователя
  - Архитектура системы (Python+Pyramid, PostgreSQL+PostGIS)
  - REST API для разработчиков
  - Примеры и видеоуроки (YouTube/RuTube)
  - Все форматы экспорта данных (GeoJSON, CSV, Shapefile, DXF, KML, KMZ)
  - Управление пользователями и система прав
  - Интеграция с QGIS, мобильные приложения
- Доступность: Полностью открыта на docs.nextgis.ru
- Используемые стандарты: OGC (WMS, WFS, TMS)

### 2. TACL Pro (версия 1.10, 2022)
**Статус документации: ПОЛНЫЙ**
- Объем: 20 страниц PDF "Руководство по установке и настройке"
- Язык: Русский
- Содержит:
  - Системные требования (Docker, Linux, PostgreSQL 14+)
  - Описание всех параметров config.json
  - Настройка БД (PostgreSQL, Oracle)
  - Авторизация (RADIUS или DB-based)
  - Внешние источники данных (postgres_fdw, oracle_fdw)
  - Полная схема внешних таблиц в приложении
  - Техподдержка: support@ksoft.pro +7(473)260-66-47
- Особенность: Описаны все 15+ типов внешних таблиц (opt_fdw_*)

### 3. Naumen Inventory
**Статус документации: ПОЛНЫЙ**
- Объем: Полная архитектурная документация
- Язык: Русский
- Содержит:
  - Архитектура платформы Naumen Kernel
  - Описание модульной структуры
  - Модель данных TMF SID
  - Технологический стек (J2EE, Tomcat, Hibernate)
  - Поддерживаемые СУБД (PostgreSQL, Oracle, MS SQL)
  - Интеграция через SOAP веб-сервисы
  - REST API для разработчиков
  - Описание ролей (системный администратор + технолог)
  - Требования к квалификации специалистов
- Особенность: Документирована полная гибкость и масштабируемость

### 4. YADRO NMS
**Статус документации: ПОЛНЫЙ**
- Объем: 2 документа - Инструкция по установке + Руководство по эксплуатации
- Язык: Русский
- Последнее обновление: 2025-11-27 и 2025-10-07
- Содержит:
  - Установка образа ПО через Docker/Kubernetes
  - Конфигурация веб-интерфейса
  - Работа с элементами сети, тревогами, конфигурациями
  - Интеграция различных типов сетевых элементов
  - Адаптация для LTE/5G сетей
  - Документация на: mpo-docs.yadro.com
- Особенность: Специализирована для операторского класса оборудования

### 5. NextGIS Mobile (версия 2.5 от 22-10-2024)
**Статус документации: ПОЛНЫЙ**
- Объем: 148 страниц PDF
- Язык: Русский + английский
- Содержит:
  - Системные требования Android 5.0+
  - Пользовательский интерфейс
  - Редактирование векторных данных
  - Работа с пользовательскими формами (NGFP)
  - API для разработчиков
  - Импорт/экспорт GeoJSON, Shapefile
  - Работа в online и offline режимах

## Решения с ОГРАНИЧЕННОЙ документацией

### Fiberbase
**Статус документации: НЕ НАЙДЕНА**
- Общедоступная документация отсутствует
- Информация только на сайте fiberbase.ru (описание услуг)
- Для получения документации: требуется обращение в компанию

### "Умный оптический транспорт" (УОТ)
**Статус документации: КОММЕРЧЕСКАЯ**
- Проект на стадии разработки (завершение июнь 2027)
- Совместный проект Ростелеком + Т8 + РФРИТ
- Документация: только под NDA
- Апробация: на сетях Ростелекома

### NMS Титан (Т8)
**Статус документации: КОММЕРЧЕСКАЯ**
- Коммерческое решение от компании Т8
- Документация: ограничена, по запросу
- Используется: в проекте Metro DWDM (ЭР-Телеком)

## РЕКОМЕНДАЦИЯ ДЛЯ ВЫБОРА

**Для максимальной доступности документации:**
1. **NextGIS** - открытая документация, community-ориентированное
2. **TACL Pro** - полная документация для установки и конфигурации
3. **Naumen Inventory** - для коммерческих операторов (требуется техподдержка)

**Для специализированных нужд:**
- Операторский класс: YADRO NMS
- Мобильный сбор данных: NextGIS Mobile
- Интеграция: NextGIS Web REST API + TACL Pro внешние источники

## ВЫВОДЫ

**Для максимальной простоты реализации аналога:**
1. NextGIS Web + Pyramid (Python) - ОПТИМАЛЬНЫЙ ВЫБОР
2. TACL Pro + Docker - альтернатива для более простых сценариев
3. Naumen Inventory - только для коммерческих операторов с техподдержкой

**Для изучения и разработки собственного решения используй:**
- NextGIS Web документацию как образец
- REST API примеры из docs.nextgis.ru
- PostGIS официальную документацию для геопространственных операций
- Pyramid фреймворк документацию для бэкенда
